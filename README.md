# Telecom Churn Prediction 

##  Project Overview
This project predicts customer churn using machine learning techniques. This project focuses on understanding customer churn behavior in the telecom industry. The goal is to identify the factors that influence customers to leave, build predictive models that estimate churn probability, and create actionable churn risk scores that telecom companies can use for targeted retention strategies

##  Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning models (Logistic Regression, Random Forest, etc.)
- Model evaluation and comparison

##  Tech Stack
- Python
- jupyter notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

##  Dataset
Telecom customer dataset with features like tenure, charges, and services.

## Results
Gradient Boosting Achieved good accuracy in predicting customer churn and identified key factors influencing churn.
